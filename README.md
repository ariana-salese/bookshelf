# Bookshelf Store

This page is an ecommerce for selling books.

## Tools

The tools being used are:

- [ReactJs](https://reactjs.org/)
- [chakra-ui](https://chakra-ui.com/)

## Run locally

Clone the repository and run:

```
npm install
npm run dev
```
